Ce package fournit les moteurs open source suivants:

Stockfish 10

Stockfish 11 20200409

Pour utiliser l'un des moteurs suivants, vous avez besoin d'une application d'échecs compatible avec le protocole Open Exchange:

Chess for All

Droidfish

Scid on the go

Chess for Android
